Sample files: 

- geospatial/ChicagoGrocery.py: find the distance to the nearest grocery store from the center of each county.
- geospatial/CobaltLikelihood.py: determine likelihood of finding Cobalt given proximity to geophysical process.
- geospatial/ML_LogisticClassifier.py: determine if a rock sample is best classified as sedimentary or metamorphic.
- geospatial/ML_TornadoRegression.py: determine how useful physical tornado characteristics are at determining damage, injuries, or death toll.

- orbital/3I-ATLAS.py: plot and propagate 3I-ATLAS trajectory using heliocentric two body propagation. 
- orbital/KeplerianOrbit.py: example of easy to use two body orbital propagation class.
- orbital/FlybyEarthInterceptKO2B.py: compute optimal intercept trajectory of 3I-ATALS to Earth. 

- sim/Traffic.py: 1D bang-bang traffic simulator. 
- sim/CompetitiveSpecies.py: numerically solve coupled non-linear differntial equations of type Lotka-Volterra.

- datascience/MahalonboisDistance.py: determine outliers in dataset.
- datascience/PELinearModel.py: parameter estimation using MCMC for linear regression coefficients. 
- datascience/RegimeSwitchDetect.py: a method to detect point of regime switch in time-series data.
